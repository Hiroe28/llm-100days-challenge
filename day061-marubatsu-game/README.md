# 🎮 マルバツゲーム

華やかなアニメーションとスコア記録機能を備えた、モダンなデザインの三目並べ（マルバツゲーム）です。

## 📸 スクリーンショット

プレイ画面のスクリーンショットを `screenshot.png` として保存してください。

## ✨ 特徴

- **美しいUI**: グラデーション背景とニューモーフィズム風のボタンデザイン
- **アニメーション効果**: 
  - タイトルのバウンスアニメーション
  - マス目配置時のポップアニメーション
  - 勝利ライン強調のグローエフェクト
  - 勝利時の紙吹雪エフェクト
- **スコアボード**: マルとバツの勝利数を記録
- **レスポンシブデザイン**: モバイルデバイスでも快適にプレイ可能
- **シンプルな操作**: クリック/タップするだけでプレイ開始

## 🎯 ゲームルール

1. マル（⭕）とバツ（❌）が交互に3×3のマス目に配置
2. 縦・横・斜めのいずれかに同じマークを3つ揃えると勝利
3. 全マス埋まっても勝負がつかない場合は引き分け
4. スコアは自動的に記録され、「もういっかい!」ボタンで次のゲームへ

## 🎨 カスタマイズポイント

### 色の変更
```css
/* マルの色 */
.cell.maru { color: #ff6b6b; }

/* バツの色 */
.cell.batsu { color: #4ecdc4; }

/* 背景グラデーション */
background: linear-gradient(135deg, #667eea 0%, #764ba2 50%, #f093fb 100%);
```

### 盤面サイズの変更
```css
.cell {
  width: 100px;   /* マス目の幅 */
  height: 100px;  /* マス目の高さ */
}
```

## 💡 使い方

1. `index.html` をブラウザで開く
2. マルのプレイヤーからスタート
3. 空いているマス目をクリック/タップして配置
4. 勝利条件を満たすと自動判定
5. 「🔄 もういっかい!」ボタンで次のゲームへ
