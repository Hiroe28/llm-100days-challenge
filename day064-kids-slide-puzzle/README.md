# 🧩 こどもスライドパズル

子供向けに設計されたインタラクティブなスライドパズルゲームです。かわいい動物や自然の写真を使って、楽しくパズルに挑戦できます。

## ✨ 特徴

- **3段階の難易度**: かんたん(3×3)、ふつう(4×4)、むずかしい(5×5)から選択可能
- **4種類のプリセット画像**: ねこ、いぬ、はな、しぜんのかわいい写真
- **カスタム画像対応**: お気に入りの写真をアップロードして遊べる
- **子供にやさしいUI**: 大きなボタンと分かりやすい日本語表記
- **完成時のお祝い**: クリア時にアニメーションとメッセージでお祝い
- **移動回数カウント**: 何手でクリアできたか記録

## 🎮 遊び方

1. **パズルのサイズを選ぶ**: かんたん・ふつう・むずかしいから選択
1. **画像を選ぶ**: ねこ、いぬ、はな、しぜんのボタンをクリック、または自分の画像をアップロード
1. **「あたらしくはじめる」をクリック**: パズルがシャッフルされます
1. **ピースを動かす**: 空いているマスに隣接するピースをクリックして移動
1. **完成を目指す**: 正しい順番に並べるとお祝いメッセージが表示されます！

## 🛠️ 技術スタック

- HTML5
- CSS3 (Grid Layout, Flexbox, Animations)
- Vanilla JavaScript
- Unsplash API (フリー画像)
- FileReader API (カスタム画像読み込み)
